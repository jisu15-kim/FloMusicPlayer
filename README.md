# 🚀 FloMusicPlayer

### 구현 기능
- 네트워크에서 받은 데이터 기반 작동
- 가사 타이밍별로 자동 스크롤/하이라이팅
- 가사 뷰에서 탭했을 때 해당 부분으로 이동

### 사용 프레임워크/라이브러리
- AVFoundation: AVPlayer 사용
- Snapkit: 오토레이아웃
- RxSwift: Reactive Programming
- Alamofire, Kingfisher
